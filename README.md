# Databáze memů a random vtipů ✨

Tato databáze bude obsahovat vtipy a memy z různých zákoutí a skulin internetu
Každá položka databáze bude mít atributy:
- První doložeé datum objevení memu/vtipu
- Kdy se nejvíce používal
- Hodnocení uživatelů (pokud nějaké získám :( )
- Odkaz na obrázek/vložený obrázek (pokud ke vtipu je)
- ? Nějaké hashtagy, ať se dá hledat podle typu


Uživatelé budou moct nahrát svůj vlastní meme/vtip, který pokud projde moderací (ať neporušuje silně nějaký etický kodex), tak bude přidán do databáze. Také se bude vyhlašovat největší přispěvatel do komunity.
